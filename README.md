# 📊 India CPI Inflation Analysis (Excel)

## 📌 Overview

This project presents an **Excel-based analysis** of India's **Consumer Price Index (CPI)** data to understand inflation trends across various expenditure categories. The analysis focuses on identifying key contributors to inflation, evaluating year-over-year and month-over-month changes, and assessing the impact of major economic events using Microsoft Excel.

---

## 🎯 Objectives

- Analyze the contribution of broader CPI categories to overall inflation.
- Study **Year-over-Year (YoY)** inflation trends for Urban and Rural CPI from **2017 onwards**.
- Identify the year with the highest inflation and its key drivers.
- Examine food inflation trends for the **12 months ending May 2023** using **Month-over-Month (MoM)** analysis.
- Evaluate the impact of the **COVID-19 pandemic** on inflation across major expenditure categories.
- Investigate the relationship between **imported crude oil price fluctuations (2021–2023)** and CPI inflation using correlation analysis.

---

## 📂 Dataset

The analysis uses monthly **Consumer Price Index (CPI)** data published by the **Government of India**, which includes:

- General CPI
- Urban CPI
- Rural CPI
- Category-wise CPI indices
- Monthly inflation values

> **Note:** Missing values were handled using appropriate Excel-based imputation techniques where required.
